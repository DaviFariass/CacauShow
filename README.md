## 📖 Sobre o Projeto

Este projeto representa o trabalho prático interdisciplinar de conclusão de semestre. Desenvolvido em colaboração real com uma franquia da **Cacau Show**, o objetivo foi criar uma solução web funcional que simula os aspectos principais de um e-commerce e de um sistema de gerenciamento de produtos.

Este trabalho demonstra a aplicação prática de conceitos de desenvolvimento web **Full Stack** e modelagem de banco de dados aplicados a um cenário de negócio real e com necessidades palpáveis.

---

## ⚙️ Tecnologias Utilizadas

O ecossistema do projeto foi construído utilizando as seguintes ferramentas:

*   **Back-end:** `PHP` - Responsável por toda a regra de negócio, lógica de servidor e processamento seguro dos dados.
*   **Front-end:** `HTML5` e `CSS3` - Estruturação semântica e estilização para garantir uma interface de usuário clara e responsiva.
*   **Banco de Dados:** `MySQL` - Modelagem e armazenamento relacional estruturado das informações de produtos e usuários.

---

## 🎯 Funcionalidades Desenvolvidas

A aplicação abrange fluxos essenciais para o gerenciamento da franquia:

- [x] **Página de Login:** Autenticação e controle de acesso para usuários já cadastrados.
- [x] **Página de Cadastro:** Formulário para registro e inserção de novos usuários no sistema.
- [x] **Gerenciamento de Produtos:** Painel dinâmico que exibe e gerencia a lista completa de produtos disponíveis na franquia.
